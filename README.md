# fastLED-WeatherClock
Simple fastLED arduino code for Wall clock showing Time, Date, Temperature and Humidity. WS2812b matrix 8x32.  
Tested with ESP01S - 1M, D1 Mini, 160MHz - 80MHz both stable in long run  
Tested with ESP32c6
